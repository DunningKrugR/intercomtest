Setup

  1. Using Basic JavaScript snippet (no build required)
  2. Enable Messenger for Web in Intercom settings
  3. Turn on user traffic in Settings → Channels → Messenger → Install
  4. Add localhost to trusted domains (if using trusted domains)
  5. Open this file directly or run npm run dev

Connection failed?

  1. Disable ad blockers – they often block Intercom
  2. Check the browser console (F12) for errors
  3. Look for a ping request in the Network tab – if missing, the script isn’t loading
